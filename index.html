LadMibs(Working Title)

Description

A community-based Mad-Libs fill-in-the-blanks game where users must fill in a single blank of a sentence. The user can invite friends to fill out another sentence. Once the game is filled, users can view the collaborated story they helped write

Design

Technologies

  • Java
  • HTML Web
  • Spring
  • Maven
  • Jackson databases

Use Cases

  • The user can join a story
  • The user can customize a random sentence in the story
  • The user can invite friends via email (or social media?)
  • The user can view completed stories completed by them or other users
